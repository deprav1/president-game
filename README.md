# Варония - Президентский Симулятор

Карточная сатирическая игра в стиле Reigns: игрок становится президентом вымышленной Варонии и пытается удержать баланс между элитами, силовиками, населением и Западом. Один экстремум по любой шкале заканчивает правление, но игра даёт специальные развилки, второй шанс и несколько финалов.

## Что уже есть

- Большая колода решений: базовые карты, дополнительные карты и отдельные карты VPN «Наружу».
- 4 ключевые шкалы: олигархи, силовики, население, коллективный Запад.
- Кризисы, сюжетные цепочки последствий и события за экстремальные значения шкал.
- Выборы раз в 4 игровых года.
- Победа после двух сроков и 10 нарративных финалов.
- Достижения, личная доска почета с топом правлений, лучший результат, хроника открытых финалов.
- Telegram Mini App UX: fullscreen, safe-area, haptics, share-ссылки.
- Сохранение прогресса через Telegram CloudStorage с fallback в localStorage.
- Интеграция «Наружу»: UTM-разметка, CTA на картах, хаб, промокоды и VPN-ревайв.

## Стек

- React 19
- Vite 8
- Telegram Web App SDK
- GitHub Actions
- Timeweb static hosting через SSH deploy

## Запуск

```bash
npm install
npm run dev
```

## Проверки

```bash
npm run lint
npm run build
```

После крупных визуальных или игровых правок стоит проверить минимум: онбординг, обычную карту, свайп/тап, кризис, поражение, VPN-ревайв, выборы, победу и рестарт.

## Деплой

Актуальный деплой идёт из ветки `main` через [.github/workflows/deploy-timeweb.yml](.github/workflows/deploy-timeweb.yml):

1. `npm ci`
2. `npm run build`
3. загрузка `dist/` на Timeweb через SSH в `./public_html/`

В `vite.config.js` используется `base: './'`, чтобы один билд корректно работал с корня домена и из подпапки. Перед заливкой workflow очищает `public_html`, чтобы удалять старые файлы и стандартные заглушки хостинга.

## Структура

```text
src/
  App.jsx                     - контейнер состояния, игровой цикл, роутер фаз
  App.css                     - дизайн-токены, layout, адаптив и основные стили
  main.jsx                    - точка входа
  components/
    GameCard.jsx              - основной экран карточки
    GameOverScreen.jsx        - поражение, промокод, VPN-ревайв
    VictoryScreen.jsx         - финал победы
    HubOverlay.jsx            - личное дело, рекорды, достижения, VPN-хаб
    LeaderboardList.jsx       - личная доска почета и топ правлений
    ElectionScreen.jsx        - выборы
    SecondChanceScreen.jsx    - второй шанс после экстремума шкалы
  data/
    cards.js                  - базовая колода, кризисы, выборы
    extraCards.js             - дополнительные карты
    naruzhuCards.js           - карты «Наружу»
    chains.js                 - отложенные последствия решений
    endings.js                - финалы победы
    params.js                 - шкалы и тексты поражений
    advisors.js               - советники и аватары
    achievements.js           - достижения
  lib/
    assets.js                 - versioned asset URLs для cache busting
    gameHelpers.js            - чистые игровые хелперы и валидация сейва
  utils/
    telegramStorage.js        - CloudStorage/localStorage обёртка
public/
  images/                     - игровые WebP/PNG изображения
docs/
  *.md                        - планы, гайды и журнал решений
```

## Правила контента

- Тексты карточек и диалоги пишутся на русском языке.
- Варония - вымышленная страна и сатирический политический аналог современной России.
- Сатира строится на узнаваемых институтах, паттернах и дилеммах, без необходимости буквально пересказывать новости.
- Новые изображения желательно сразу хранить как `.webp` в `public/images/`.
