# 🦅 Варония — Президентский Симулятор

Карточная игра в стиле Reigns, где ты — президент вымышленной страны Варония. Принимай решения, балансируй между элитами, армией, народом и Западом. Один неверный шаг — и тебя свергнут.

## 🎮 Геймплей

- **115+ карт решений** — экономика, армия, дипломатия, внутренняя политика
- **8 советников** — от министра финансов до олигарха
- **Кризисные события** — теракты, военные конфликты, дефолт
- **Цепочки последствий** — решения аукнутся через месяцы
- **Президентские выборы** каждые 4 года
- **Свайпы и тапы** — мобильный интерфейс

## 🛠️ Стек

- React 19 + Vite 8
- Telegram Web App SDK
- Деплой: Vercel → Telegram Mini App

## 🚀 Запуск

```bash
npm install
npm run dev
```

## 📦 Билд

```bash
npm run build
npm run preview
```

## 🤝 Для коллег

1. Форкни репо
2. `npm install`
3. `npm run dev` — запусти локально
4. Создай ветку: `git checkout -b feature/твоя-фича`
5. Пушь и создай PR
