# План оптимизации и стилевой доводки «Варония»

Документ описывает приведение всего проекта к единому стилю портретов —
**«имперский минимализм / тлен»** (чистый чёрный фон, плоский вектор, золото,
сарказм, упадок империи) — плюс оптимизацию производительности и кода.

Статус на момент написания: тема уже переведена в чёрно-золото, портреты
интегрированы, картинки сжаты в WebP (52 МБ → 1.9 МБ), деплой на Timeweb работает.

---

## Принципы стиля (эталон — портреты советников)
- **Фон:** чистый чёрный до краёв, без рамок.
- **Палитра:** золото `#d4af37`, тёплый чёрный `#0a0603–#16100a`, оксблад для кризиса.
  Акценты фракций (красный/зелёный/синий) — только в иконках/статусах.
- **Форма:** плоский вектор, минимум деталей, крупный объект + негативное пространство.
- **Тон:** сарказм, державный пафос с трещиной, «тлен».
- **Типографика:** Playfair (лого) + Inter (UI) + JetBrains Mono (досье/чипы).
