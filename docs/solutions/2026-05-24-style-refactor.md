# Рефакторинг стилей и мобильная адаптация (Telegram WebApp)

## Problem
Игра представляла собой монолит с огромным объектом инлайн-стилей `STYLES`. Это делало код нечитаемым, затрудняло добавление новых элементов и делало невозможным использование CSS-переменных. Кроме того, шрифты (печатная машинка) были плохо читаемы на мобильных устройствах, а интерфейс перекрывался системными элементами iOS/Android (челки, свайперы).

## Root Cause
- Изначальная архитектура базировалась на инлайн-стилях для быстрого прототипирования.
- Игнорирование `env(safe-area-inset-*)` для мобильных браузеров и Telegram WebApp.
- Использование стилизованного, но непрактичного шрифта `Special Elite` для мелких UI-элементов.

## Solution
1. **Вынос стилей в CSS:** Создан `App.css` с корневыми CSS-переменными (`:root`) для цветов и шрифтов. Все глобальные стили, анимации и специфичные классы (например, `.card-advisor-avatar`) перенесены туда.
2. **Шрифтовая реформа:** `Special Elite` полностью заменён на `PT Mono` (`--font-typewriter`) — моноширинный шрифт, который корректно рендерится без отдельной загрузки на большинстве платформ и хорошо читается в малых размерах. Художественный текст карт и заголовки по-прежнему используют `Playfair Display` (`--font-serif`). Системный гротеск (`--font-sans`) применяется для кнопок и мелких UI-подписей.
3. **Safe Areas:** В корневой контейнер `App.jsx` добавлены `padding-top: env(safe-area-inset-top)` и `padding-bottom: env(safe-area-inset-bottom)` через CSS-класс `.app-shell`.
4. **Адаптивные размеры:** Аватары советников теперь масштабируются через `min(20dvh, 125px)`, что предотвращает поломку верстки на старых iPhone.
5. **SVG-иконки фракций:** Растровые PNG-иконки (`icon_oligarchs.png` и т.д.) заменены инлайновыми SVG через компонент `FactionIcon`. Это исключает 404-ошибки на иконках и устраняет прежний баг с выводом пути файла вместо картинки.

## Prevention
- Впредь любые глобальные цвета и новые шрифты добавлять только через CSS-переменные в `App.css`.
- Использовать инлайн-стили только для динамических вычислений (например, ширина `StatPill` или трансформации свайпа).
- Для иконок фракций всегда использовать `FactionIcon type={p.key}`, не растровые изображения.
- Любые новые экраны тестировать с включенным симулятором Safe Areas.
