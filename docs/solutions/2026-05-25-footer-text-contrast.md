# Решение: Низкий контраст текста «Наружу» в футере на светлом фоне

## Problem
Слово «Наружу» в футере главного меню и экранов настроек было практически нечитаемым для пользователей. Текст сливался со светлым пергаментным фоном (light beige) панели меню.

## Root Cause
Для окраски слова «Наружу» в футере использовалась константа `NARUZHU_YELLOW` (`#FFD60A`), которая представляет собой фирменный неоново-желтый цвет бренда. Этот цвет разработан для использования на темных и стеклянных (dark glassmorphic) фонах, например в «Наружу Hub», где он выглядит отлично и имеет высокую контрастность. Однако при рендере на светлом пергаментном фоне панели (`#f5e6c8`) этот цвет создавал крайне низкий коэффициент контрастности (~1.1:1), делая текст нечитаемым и нарушая требования доступности (WCAG).

## Solution
Вместо использования глобальной константы `NARUZHU_YELLOW` для инлайн-стиля в футере (`src/App.jsx`), цвет слова «Наружу» был изменен на высококонтрастный темно-янтарный цвет (`#b45309`). 
Этот цвет:
1. Имеет отличный коэффициент контрастности (~4.13:1) на пергаментном фоне, что делает его абсолютно читаемым.
2. Сохраняет ассоциацию с фирменным желтым/золотым брендом «Наружу».
3. Гармонично вписывается в общую цветовую гамму игры (темно-золотые и коричневые тона).

В файле `src/App.jsx` произведена замена:
```diff
- 🚪 Игра от <b style={{ color: NARUZHU_YELLOW }}>Наружу</b> — надёжный VPN для свободного интернета
+ 🚪 Игра от <b style={{ color: "#b45309" }}>Наружу</b> — надёжный VPN для свободного интернета
```

## Prevention
1. **Разделение тем**: При использовании брендовых цветов (особенно ярких неоновых, таких как желтый, зеленый или голубой) всегда учитывать текущую тему (светлая/темная) или фоновый элемент.
2. **Контроль контрастности (WCAG)**: Для текстов на светлых фонах использовать оттенки желтого/оранжевого не светлее темно-янтарного (`#b45309` или `#854d0e`), чтобы контрастность составляла не менее 4.5:1 (или хотя бы 3:1 для жирного/крупного шрифта).
3. **Локальные переопределения**: Не использовать константы, предназначенные для темных тем, на светлых элементах интерфейса без адаптации цвета или подложки.
